思想汇报

尊敬的党组织：

　　近期，经过相关课程的学习，在理论思想、党性修养、政治生活等方面获益匪浅，深刻感受到作为一名积极分子（预备党员）所要肩负的责任与使命，立志以工匠精神耕耘党建。

　　9月10日，全国教育大会在北京召开。中共中央总书记、国家主席、中央军委主席习近平出席会议并发表重要讲话。他强调，在党的坚强领导下，全面贯彻党的教育方针，坚持马克思主义指导地位，坚持中国特色社会主义教育发展道路，坚持社会主义办学方向，立足基本国情，遵循教育规律，坚持改革创新，以凝聚人心、完善人格、开发人力、培育人才、造福人民为工作目标，培养德智体美劳全面发展的社会主义建设者和接班人，加快推进教育现代化、建设教育强国、办好人民满意的教育。

　　我对于习总书记的重要讲话，如何将教育、工匠精神运用到党建中去，有了自己的理解，我认为要做好三个方面：

　　第一，学好党章党规，坚定理想信念。

　　回顾党章曲折辉煌的发展历史，全心全意为人民服务是贯穿党章的鲜活灵魂。作为积极分子（预备党员），要在政治上与党中央保持一致，认真学习中国特色社会主义理论，不断提升自己的党性修养，以党员的标准严格要求自己。给自己留下深刻印象是习总书记强调，长期以来，广大教师贯彻党的教育方针，教书育人，呕心沥血，默默奉献，为国家发展和民族振兴作出了重大贡献。自觉用中国特色社会主义理论体系武装头脑，要兼有世界眼光、中国情怀、时代特征，让青春在奋斗中焕发绚丽的色彩。笃实，扎扎实实干事，踏踏实实做人。道不可坐论，德不能空谈。于实处用力，从知行合一上下功夫，核心价值观才能内化为人们的精神追求，外化为人们的自觉行动。

　　第二，提升党性修养，培育践行社会主义核心价值观。

　　我将牢记习总书记提出教育是民族振兴、社会进步的重要基石，是功在当代、利在千秋的德政工程，对提高人民综合素质、促进人的全面发展、增强中华民族创新创造活力、实现中华民族伟大复兴具有决定性意义。努力做到“三个一”：保持第一标准，希望别人相信的要自我坚信；牢记第一身份，守好政治纪律和政治规矩；坚守第一生命线，党内生活特色不褪色。党的十八大以来，习近平总书记多次强调坚持党要管党、从严治党，明确提出要严肃党内政治生活，用好批评和自我批评这个利器。作为一名积极分子（预备党员），我认为可以利用支部这个平台，定期向支部介绍人汇报思想，开展成员的批评与自我批评。只有经常地开展批评和自我批评，虚心地接受批评才能不断地改造自我，通过总结得失找到自己的优点和不足，提高自我，完善自我。在批判式教学中培养自己看待问题的辩证智慧，学会批判、思辨、辩证，使自己养成面对现实问题时，冷静、客观地分析问题和解决问题，学会历史地、辩证地面对人生。在一次次地批评与自我批评中考虑问题、解决问题，并将此类技能运用到学习生活以及工作中去，将生活从单一变为丰富，高行动标准，提升实践效果。

　　第三，开阔视野提升境界，不断增强为民服务的本领。

　　提升为民服务本领就要强化学习，认真贯彻落实“三严三实”要求，塑造良好形象强化人格魅力。“书中自有黄金屋”，我们应该强化工匠精神，经常开展政治学习形式的分享交流，一方面通过互相监督互相引导，保证学习交流的按期完成，促进理论学习；另一方面通过分享和交流，能够互相学习，并且进行深入的思想探讨，以此来落实中国共产党“名言思辩、讨论学习”的优秀精神。除了倡导学习交流形式的理论学习培养每一位积极分子（预备党员）的理论水平和学习能力之外，更注重“知识产出”，通过学习讨论和实践真知，以多种形式督促并产出自己的知识积累。做到外化于行，内化于心，不断创新，更新观念，大胆尝试和探索，真正做到领悟大情，感知大义，方能运用到学习生活和工作中去。

　　始于匠心，忠于党心。只有不断强化党内生活的政治性、原则性、战斗性，党员干部的党性才会越来越坚强；只有做到严以修身、严以律己，才能使我们的党始终保持凝聚力、向心力；只有把理想信念和党性观念摆在首位，不断增强政治意识，严守政治纪律，强化政治担当，才能使党内政治生活在全党严肃认真开展起来。

　　养浩然之气，拥优良党风，塑忠诚党心，终而造诣精深，成其大器。

　　此致

敬礼！
