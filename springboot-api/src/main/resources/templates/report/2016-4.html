思想汇报

尊敬的党组织：

　　在“两学一做”学习教育如火如荼开展之际，习近平总书记发表了纪念中国共产党成立95周年重要讲话，向全党发出了“不忘初心、继续前进”的时代口号。“不忘初心、继续前进”，实现“两个一百年”的奋斗目标，需要牢记服务宗旨的党员干部的坚实支撑，需要有“四讲四有”合格共产党员的奋力践行。作为入党积极分子，又递交了入党申请书，面对新形势新要求更应该主动发挥表率作用，当好标兵、作出榜样，让党员旗帜高高飘扬。

　　“知者行之始，行者知之成”。党员只有心有定见，才能本正形立，才能在学习、生活中树标杆、做表率。作为一名学习积极分子，必须通过“学思践悟”，用实际行动来诠释践行“四讲四有”，以一名合格党员的标准严格要求自己，争取早日成为一名合格的党员。

　　一、思想上，更加明白了担当的内涵

　　在思想上，我不断加强思想政治学习，除了自觉学习毛泽东思想、邓小平理论、“两学一做”等重要思想外，我还经常阅读有关党建、党史及党员先进事迹的书籍等，不断加强对党的路线、方针政策的认识和理解，使本人在学习生活上有更大的精神动力，做到学以致用，树立更加正确的人生观、价值观和世界观，自觉抵制腐朽思想和不正之风的侵蚀。在学习习近平讲话精神时，我看到习近平总书记多次强调，党员要有担当精神，敢于作为。担当和责任是一种精神、是一种境界。敢担当，首先需要我们舍我其谁的魄力、争创一流的意识、奋发有为的精神状态。作为学生，要自觉把学习当事业干，把专业课学习当阵地守，从我做起，从现在做起，真正彰显共产党员敢于担当精神和勇气，要把责任与担当作为精神坐标。

　　二、生活上，要发挥带头作用

　　践出干劲，就要发挥带头作用，用奉献写就对人民的大爱。践是思的检验。纸上得来终觉浅、绝如此事要躬行。“讲奉献、有作为”作为做合格党员的落脚点，使每一个党员就像一面旗帜高高飘扬，引领方向，激励上进。讲奉献、有作为是为了集体利益，发自内心地去为人民和社会做一些事情。在生活上，我注意从平时做起，从小事做起，不怕苦不怕累，脚踏实地，认真工作。在平凡的工作岗位上做一个不平凡的人，努力以全心全意为人民服务的理念切切实实地为身边的同事、为组织、为国家做点实事。

　　三、踏上工作岗位后，要把敬业作为今后的准则

　　学好精神，就要增强敬业意识，用觉悟写就对事业的挚爱。“忠诚印寸心，浩然充两间。”“两学一做”活动作为党员爱岗敬业、拼搏奉献的助推器，每一名党员都要深学细照笃行，发挥先锋模范作用，立足岗位践行对党的事业的挚爱。我们要通过学习，把对党、对人民的感恩之情转化为坚定信念、听党指挥的崇高追求，转化为爱岗敬业、无私奉献的高尚精神，转化为埋头苦干、服务群众的实际行动，转化为勇创佳绩、拼搏进取的工作劲头。只有以服务企业群众为己任，一切为了人民，一切为了人民的物质财产安全，才能热爱本职工作，把业务做精、做细、做好，用真挚的服务赢得人们发自内心的尊重。虽然我只是学生，但是今后也会踏上工作岗位。今后要继续牢记宗旨意识，加强道德修养，切实做到学而信、学而行。踏上工作岗位后，既要做好本职工作，把服务企业群众作为孜孜以求的目标；更要学好做人，遵守职业道德。从现在做起，从小事做起，积极主动投身到民生保障的伟大事业中，争做让党放心、让群众满意的好党员。

　　今后，我要不断经受党的考验，在学习中以高标准、严要求的党员态度去勇于担当、甘于挑战，认真开展并长期坚持“两学一做”学习教育，在自己的专业学习上发光发亮。

　　此致

敬礼！
