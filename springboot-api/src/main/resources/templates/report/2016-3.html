思想汇报

尊敬的党组织：

　　成为积极分子已经有段时间了，在这段日子里，我时刻牢记自己是一名入党积极分子，更进一步的理解党的基本路线、方针、政策等的真正含义，使我在生活、学习、思想等各方面都有了更加明确的目标和更强大的动力。十八大以来，党所取得的瞩目成绩，既是由于以习近平同志为总书记的党中央的正确领导，同时也离不开千千万万名党员同志，特别是基层一线党员干部的努力进取分不开的。在前不久召开的G20杭州峰会上，世界聚焦中国。我作为一名入党积极分子，不仅感到光荣，而且责任重大，中国国家主席习近平总书记将这次G20峰会的主题定为“构建创新、联动、活力、包容的世界经济”，实际上是在总结中国经济发展的现有模式之基础上，对中国与世界经济提出的既现实、又前瞻的远景目标。现将我学习G20峰会精神之后的思想汇报如下：

　　一、思想上，明白创新与发展是民族重任

　　G20峰会上，习近平总书记说到中国的发展始终离不开一个“新”字，新起点、新常态，就是始终保持着创新的动力与韧劲。习近平总书记说，“抓创新就是抓发展，谋创新就是谋未来”。然而创新与发展，并不纯粹是技术层面的事。创新、发展、未来，最终指向的是人民群众的获得感，是中华民族的复兴，是13亿中国人全面进入小康社会的富民之路，强国之路。回顾建国初期，“两弹一星”的发展经历了我国多年来的自主研发、三年的经济困难时期、攻克三大技术难关等阶段最终成为现实，我国在一穷二白的条件下，独立研制成功原子弹、氢弹和人造地球卫星。这种从无到有、敢于亮剑、大胆创新的精神正是目前迎接市场压力的重要动力。在今天，这种创新精神更加难能可贵，我们学生是祖国的未来，更应该培养自己的创新能力。

　　二、政治上，明白共产党的政治重任

　　中国的创新发展，首先是中国共产党所担当和坚持的政治重任。中国共产党建党95年来，带领人民所走的道路，不论在哪个阶段，都不忘初心。这个初心就是，全心全意为人民服务，实事求是，与时俱进。历史发展到现阶段，中国在落实各项建设的决策过程中所做的是，坚持问题导向，破解发展难题，始终维护最广大人民群众的根本利益，不断开拓中国特色社会主义的新境界。这是中国经济发展的根本性动力，也是创新发展的最大动力。中国的经济能够担当世界经济发展的引擎和平衡仪，正是基于中国共产党和中国人民所选定的这条政治道路。这是道路优势的魅力所在。

　　三、学习上，明白当前任务是狠抓学习

　　通过这次对G20峰会的关注和学习，我更加深刻的认识到了自己作为一名当代青年，身上的历史使命与责任。中华民族是一个充满着朝气的民族，新时期的中国是我们实现梦想的时代。作为当代青年，尤其作为一名入党积极分子，我的梦想与国家民族兴衰荣辱紧密相连，我会勇敢追求自己心中的梦想，并付诸行动，加强学习，为实现“中国梦”增添新的活力。新时代的中国正面临着改革开放的关键时期，社会各个领域都需要党员去维护、去建设。新中国也正等着青年们去改革、去拉动。这就促使我们学生必须争做一名合格的共产党员，努力搞好学习，提升自身党性修养，与时俱进，以便将来不断迎接新形势下的新挑战。

　　在今后的学习生活中，我会坚定自己正确的人生观与价值观，热爱党，热爱祖国，热爱人民，坚决拥护党的路线方针政策，忠实践行科学发展观，对坚持走中国特色社会主义道路、实现全面建成小康社会的宏伟目标充满信心。我会不断努力的提高自身的政治素质，加强学习与社会实践，请党组织考验我吧。

　　此致

敬礼！
