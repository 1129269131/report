思想汇报

尊敬的党组织：

　　时光匆匆，白马过隙，转眼间2018年就要过去了。这一年来，在领导的关心和同事们的帮助支持下，思想上，我始终严格要求自己，向身边优秀党员学习，凡事身先士卒，吃苦在前，积极学习党的理论知识，端正入党动机；学习上，我始终勤奋学习，积极进取，不断努力提高自我；生活上，我乐观开朗，热情大方，积极面对生活。这一年是付出的一年，是充实的一年，是收获的一年，也是进步一年。这一年，在组织的关怀下，我成功被吸收为积极分子（发展对象、预备党员），我通过不断学习，在政治理论、文化素质各方面都有了很大收获 ，现将个人一年来的思想汇报如下：

　　一、思想上，加强政治学习，不断提高政治思想觉悟

　　我不断加强政治学习，积极参加党支部组织安排的各种政治学习和思想教育活动，认真学习思想政治教育，不断改造自己的世界观、人生观和价值观，理论联系实际，在工作中实践、实践中总结、总结中提高进步。用习近平系列重要讲话精神、十九大精神来指导自己做好自己的世界观、人生观和价值观，增强自己的政治敏锐性和辨别是非的能力，深入践行“两学一做”学习教育活动，树立牢固的共产主义信念和全心全意为人民服务的宗旨意识。自被吸收为积极分子（发展对象、预备党员）以来，我努力学习党的基本理论和重要思想，努力做到在思想上、认识上同党中央、党支部保持高度一致、坚定不移执行党的决策与部署，坚定不移听党话跟党走。

　　二、学习上，努力提高自身政治素养和综合素质

　　自被吸收为积极分子（发展对象、预备党员）以来，我认真学习了《中国共产党章程》、《中国共产党廉洁自律准则》、《中国共产党纪律处分条例》、《中国共产党党员权利保障条例》以及《习近平总书记系列重要讲话读本》等内容，努力在学习中读原著、学原文、悟原理。通过学习，我的思想得到极大改造，觉悟得到进一步提高，更加认识到入党的重要性与必要性，更加庆幸自己能够被接收积极分子（发展对象、预备党员）。除了学习党的知识，我还主动关心了国内外大事，这一年，发生了很多大事喜事， 2018年3月3日和3月5日，全国两会即“中国人民政治协商会议第十三届全国委员会第一次会议”和“中华人民共和国第十三届全国人民代表大会第一次会议”，分别于在中国北京人民大会堂开幕。博鳌亚洲论坛2018年年会于4月8日拉开帷幕，迎接来自世界各国的2000多位嘉宾。年会以“开放创新的亚洲，繁荣发展的世界”为主题，分为“全球化与‘一带一路’”“开放的亚洲”“创新”和“改革再出发”四大板块，共60多场正式讨论。5月2日，中共中央总书记、国家主席、中央军委主席习近平来到北京大学考察。习近平总书记在与北大师生座谈时发表了重要讲话，并在广大师生中引起强烈反响。习近平强调，当代青年是同新时代共同前进的一代。广大青年既拥有广阔发展空间，也承载着伟大时代使命。他指出：广大青年要忠于祖国、忠于人民；要立鸿鹄志、做奋斗者；要求真学问、练真本领；要知行合一、做实干家；要有社会主义建设者和接班人的使命担当。9月4日，习近平主席在中非合作论坛北京峰会开幕式上的主旨讲话中提出实施“八大行动”，成为中非领导人与工商界代表高层对话会暨第六届中非企业家大会上的热议话题。10月，改革开放40年引起社会各界热议庆贺。一件件大事喜事，彰显了党的先进性，彰显了民族复兴之路正在前进，更加坚定了我入党的决心与信心。

　　三、生活上，我严格要求自己，始终坚持端正的生活作风

　　生活中，我严格遵守党的纪律要求，进一步规范自己的言行，同时注重坚强自己修养，始终保持一种积极向上的生活状态，投入到每天的生活中。与同学朋友团结协作，积极向老同志学习、互相帮助、互相提高，圆满完成各项任务。

　　四、个人不足及今后努力方向

　　在2018年，我时刻要求自己从实际出发，坚持高标准、严要求，力求做到业务素质进一步提高。在工作做得细致，坚持学以致用，但还有许多不足之处：一是学习有不足之处，个人的党内知识有待进一步提高；二是在处理问题时，经验不足，不够熟练。

　　总结了过去，方能展望未来！辞旧迎新，在总结本年度思想的同时，针对自己不足之处，在新的一年要继续加强学习，继续努力，争取早日通过党的重重考验，早日成为一名正式党员，请组织继续考验我。
