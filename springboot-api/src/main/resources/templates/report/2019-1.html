思想汇报

尊敬的党组织：

　　2019年3月5日，第十三届全国人民代表大会第二次会议在北京人民大会堂开幕。据会议议程，李克强代表国务院向大会作政府工作报告。报告共分三部分：一、2018年工作回顾；二、2019年经济社会发展总体要求和政策取向；三、2019年政府工作任务。李克强从八个方面对过去一年的工作进行了回顾：创新和完善宏观调控，经济保持平稳运行；扎实打好三大攻坚战，重点任务取得积极进展；深化供给侧结构性改革，实体经济活力不断释放；深入实施创新驱动发展战略，创新能力和效率进一步提升；加大改革开放力度，发展动力继续增强；统筹城乡区域发展，良性互动格局加快形成；坚持在发展中保障和改善民生，改革发展成果更多更公平惠及人民群众；推进法治政府建设和治理创新，保持社会和谐稳定。

　　作为一名积极分子，我看到李克强总理的报告中18年的成就折射出我们国家综合国力的强大，综合国力的强大则反应出党的各种改革的成效，大到京津冀协同发展、长江经济带发展、粤港澳大湾区建设等国家战略稳步实施,小到我们日活不断得到改善，生活水平的提高，飞机、高铁、医疗、教育等基础设施保障完善，这些改善都是落实到我们的日常生活中去的，便利我们的生活，使我们的幸福指数得到了极大的提升。看到李克强总理的报告中里我国取得的成就，我感慨良多，特向党组织汇报我这段时间的学习生活情况。

　　一、思想上，树立崇高的入党动机

　　通过学习两会精神，我再次见识了我国的强大国力，以及党中央聚焦民生、改善民生的决心。取政于民，执政为民，中国共产党扎根于群众，也是服务于群众。这就对我们积极分子提出了更高的要求：要进一步端正入党动机。我曾多次在以往的思想汇报中坦露过自己的入党动机问题，从初时的随大流的错误思想，到后来在与学院领导和支部党员的谈话中，转变为想要与党内优秀同志比肩，想要更好地服务社会，让自己真正当得起“党员”二字的分量。自加入放射科党支部后，接触了党员模范，在与历届的优秀党员的交往中，我清楚地认识到自己与他们的差距不仅仅是在成绩上，更是在于言行上，折服于他们的人格魅力和气度的同时，我也想要成为像他们一样优秀的人。此后的时间里，在党组织的帮助下，我逐步改变刚参加工作时的偶尔放松的态度，树立了自己的榜样标杆，针对同志们向我提出的成绩还有待加强的意见，我一直在端正自己的工作态度，寻找适合自己的学习方法。

　　二、学习上，敢于接受考验

　　两会既是我们的行动指南针，也是我们的风向标。作为一名入党积极分子，即使我们只是学生，也可以在学习上有所表现。作为一名青年人，我们肩负着建设中国特色社会主义事业的重任，我们被时代赋予了新的历史使命。时代要求我们必须要认清当前的形势，坚定我们的共产主义信念，认真的在自己的工作、学习、生活中严格用党员的标准要求自己，在纪律作风上严格规范自己的言行，不断坚定自己的奋斗目标，在现代化建设和改革开放中用自己的实际行动为祖国和人民贡献自己的力量。

　　三、 生活中，发挥好模范作用

　　在生活中，我努力做到比贡献、比业绩，不比待遇，艰苦奋斗的作风不能丢。提倡艰苦朴素，勤俭节约，反对大手大脚。遵纪守法，接受监督，抵制腐朽思想的侵蚀。我坚信不怕苦、不怕累，敢于付出，辛苦自己，才能方便他人，谦虚慎行就一定会取得更大的成就，在生活和工作中决不做一个“口惠而实不至”的人，以行动诠释责任。

　　下一步我将继续保持勤学好问的学习状态，干中学、学中干，切实提升个人的业务素养和党性修养；积极向党组织靠拢，持之以恒用党员标准严格要求自己，自觉地接受党员和群众的帮助与监督，努力克服自己的缺点和不足；学先进、做先进，履职尽责，当好一名学生，以实际行动影响身边人，希望党组织从严要求我，帮助我成长，力争尽早加入党组织。

　　新的一年，不只是党员，每一个人都应该考虑如何把自己的生活与两会精神相结合，只有把两者相结合，才能指导一年的行动。作为我，一名学生，我会认真努力履行自己的学习职责，为将来走上社会做铺垫。

　　此致

敬礼！
