思想汇报

尊敬的党组织：

　　今年，中共中央办公厅印发了《关于在全体党员中开展“学党章党规、学系列讲话，做合格党员”学习教育方案》，并发出通知，要求各地区各部门认真贯彻执行。开展“两学一做”学习教育，是面向全体党员深化党内教育的重要实践，是推动党内教育从“关键少数”向广大党员拓展、从集中性教育向经常性教育延伸的重要举措。这段时间，我也参加了学院组织的两学一做活动。随着“两学一做”工作的全面开展，我深受启发且收获良多，对作为一名积极分子，应如何具备党员素质、履行党员义务等有了进一步的认识和定位，同时我的思想理念也得到了新的激发和升华。以下我将“两学一做”学习教育活动与自身实际工作经历相结合，向党汇报我的思想：

　　一、忠于职守，不忘初心

　　“用生命捍卫使命，以责任书写党性；在责任中坚守，在使命中前行。”这句话真真切切地道出了党的95周年历史发展长河中，有一批又一批网信办革命先辈用自己的实际行动释了“不忘初心，一心向党”的意义。作为一个学生，要时刻把责任扛在肩上，把担当握在手中，忠于职守，奋斗进取，时刻认识到学习任务的重要责任，立足本职，搞好学习。“空谈误国，实干兴邦。”为了早日成为一名正式的共产党员，我必须时刻以系列讲话中的重要内容作为武装头脑的理论基础，不忘初心。今后做事要务实，要站在大局上谋思路、想问题、出举措，争取在平凡的学习生活上作出成绩。今后要努力做好本职学习和提升自身党性修养结合起来，与时俱进，不断迎接新形势下的新挑战。

　　二、立足平凡，谱写忠诚

　　在党的发展历史进程中有许许多多为党的事业呕心沥血不断进取的先进榜样，我作为积极分子，在学习生活中也应该像先进党员“铁人”王进喜同志、“愚公”李保国同志那样无怨无悔，坚守岗位，怀有共产党员所具备的那份责任心和意志力，在名利诱惑面前能经得起诱惑、顶得住考验，用自己的实际行动来谱写无与伦比的忠诚，来演绎不平凡的人生。无论将来从事什么职业，都必须牢记党的发展历史，明白幸福生活来之不易，必须且行且珍惜。

　　三、艰苦奋斗，言行合一

　　艰苦奋斗精神是中华民族的传统美德，作为一名入党积极分子，在工作中要始终保持严谨认真的工作态度，这是做合格党员的根本出发点。在新的历史时期，无论是身处顺利或者身处逆境时都不能精神懈怠，都要始终践行“两学一做”学习教育的核心，发挥党员干部艰苦奋斗的精神，求真务实，发扬“五大优良作风”中的艰苦奋斗之风，积极献身学习之中，以饱满的热情投入到日常学习生活之中，只有这样才称得上是一名合格的党员。目向星辰，脚踏实地。习总书记在系列讲话中提到：“作为共产党员，我们一定要始终与人民心心相印、与人民同甘共苦、与人民团结奋斗，夙夜在公，勤勉工作，努力向历史、向人民交出一份合格的答卷。”“纸上得来终觉浅，绝知此事须躬行。”我作为一名积极分子，同时也是一名学生，在学习中要“一步一个脚印”，将学习过程和结果都达到尽善尽美，勤奋努力才是应有的学习态度。

　　“明镜所以照形，古事所以知今”，作为一名积极分子，通过“两学一做”学习教育活动的开展，从根本上夯实了我的政治理论基础，提高了我的党性党悟，坚定了我的政治方向，使我对立足本职、不忘初心、忠诚于党的内涵有了更深一步的认识，这不仅有利于今后为人民群众提供更好的服务，更进一步夯实了入党理论基础。我将努力坚持树立远大理想与进行艰苦奋斗的统一、坚持学习科学文化知识与加强思想道德修养的统一、坚持实现自身价值与服务祖国人民的统一、坚持学习书本知识与投身社会实践的统一。在今后的学习、工作和生活中，我会不断地进行学习消化，严格要求自己，真正提高自身思想道德修养。以上是我这段时间的思想动态，请组织考验！

　　此致

敬礼！
