思想汇报

尊敬的党组织：

　　2017年已经春意盎然，我沐浴在党的春风里，不断的学习和提高，有了很大的进步。上学期，我怀着激动的心情向党组织提交了入党申请书，通过这一段时间的学习与实践，我认识到共产党员不仅仅是一种身份，一种光荣，更加意味着一种责任、拼搏、奋斗，甚至是牺牲。在这些日子里，我不断的提高自己，充实自己，严格按照党员标准去要求自己，树立正确的世界观、人生观和价值观，坚定共产主义理想和社会主义信念，努力积极的向党组织靠拢。我现将近期的思想、学习生活情况汇报如下：

　　中共中央总书记、国家主席、中央军委主席习近平3月5日下午在参加他所在的十二届全国人大五次会议上海代表团审议时强调，解放思想，勇于担当，敢为人先，坚定践行新发展理念，深化改革开放，引领创新驱动，不断增强吸引力、创造力、竞争力，加快建成社会主义现代化国际大都市。

　　思想上，我深入的学习了马克思主义基本原理、毛泽东思想、邓小平理论和“三个代表”重要思想。在认真学习和领会党的理论、纲领、路线、方针、政策等知识的同时，力争做到从思想上时刻与党中央保持高度一致。通过学习，我不断牢固树立正确的世界观、人生观和价值观，加强自己的责任感和使命感。从我做起，从现在做起，从一点一滴做起，走在时代的前列，争做时代的先锋，是我的责任，也是我的光荣。我们大学生有能力负担起建设小康社会的使命，与时俱进，开拓创新，求真务实，为实现“中国梦”做出自己的贡献。

　　中共中央总书记、国家主席、中央军委主席习近平3月29日上午在参加首都义务植树活动时强调，植树造林，种下的既是绿色树苗，也是祖国的美好未来。要组织全社会特别是广大青少年通过参加植树活动，亲近自然、了解自然、保护自然，培养热爱自然、珍爱生命的生态意识，学习体验绿色发展理念，造林绿化是功在当代、利在千秋的事业，要一年接着一年干，一代接着一代干，撸起袖子加油干。

　　作为一名中国古代史的学生，我们深知自然环境对人类的重要性，习主席带领的植树活动，从行动上号召广大青少年保护自然环境，珍爱生命，从历史文化角度出发，濒临灭绝的动植物数不胜数，绝大多数情况还是人为的，人类与自然应该相互依赖。我国耕地、水和矿产等重要资源的人均占有量都比较低。今后随着人口增加和经济发展，对资源总量的需求更多，环境保护的难度更大。必须切实保护资源和环境，不仅要安排好当前的发展，还要为子孙后代着想，绝不能吃祖宗饭，断子孙路，走浪费资源和先污染、后治理的路子。要根据我国国情，选择有利于节约资源和保护环境的产业结构和消费方式。坚持资源开发和节约并举，克服各种浪费现象。综合利用资源，加强污染治理。

　　学习生活中，我认为，作为中国古代史的学生，我们的首要任务就是学习，在我积极向党组织靠拢的过程中，要起到先锋模范作用，除了思想表现好之外，另一个重要因素就是在古代史专业的学习上也要过硬。我认真的学习专业的每一门课程，在课堂上尽可能做好重点知识的记录，课后加以巩固，并利用课余时间阅读有关书籍，与同学一起参加一些专业以外的活动和讲座，拓展自己的知识面。除了学习专业知识，我也特别注意自己在英语和计算机能力等方面的提高，努力提升自己的综合素养。我坚信：只要自己努力，踏踏实实的学习，努力实践，并且虚心的去接受别人的指导和意见，就可以做出成绩。

　　今年是我国进入“十三五”规划的第二年，标志着一个新的时代开始了。我通过网络和电视等新闻媒体了解到“十三五”规划远不仅仅是一项国策，一个决定，它更是一种能力，是社会治理体系的不断完善和更新。“十三五”规划是一项系统工程，它不仅仅需要执政者的坚定决心，还需要从开展立法到向全民普法、再到推进执法，这当中不论哪个环节松懈了，都可能影响全局。作为一名当代大学生，我们身上的责任重大。今后，我会认真学习“十三五”规划的精神，在未来的学习和生活中努力做到将自己的个人价值和社会价值的实现统一起来，争做一名合格的共产党员。

　　作为一名中国古代史的研究生，现在的我，必须要有一个明确的目标，才有更强的动力继续奋斗。“梅花香自苦寒来，宝剑锋从磨砺出”。我要做的就是根据党员的要求不断改造自己，让自己向好的方面发展。在古代史的研究生学习期间，我会利用好每一分钟来做有意义的事，我会利用好每一次机会来锻炼自己。

　　今后，我将继续不断加强思想政治学习。坚定自己正确的人生观与价值观，热爱党，热爱祖国，热爱人民，坚决拥护党的路线方针政策，忠实践行科学发展观，对坚持走中国特色社会主义道路、实现全面建成小康社会的宏伟目标充满信心。请党组织考验我吧。

　　此致

敬礼！
