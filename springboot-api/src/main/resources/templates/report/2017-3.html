思想汇报

尊敬的党组织：

　　时值丹桂飘香，我成为入党积极分子（预备党员）已有5月之久，中国共产党因数千万党员的不懈努力而蓬勃发展，生机盎然，各共产党员因党的发展与精神而事业有成，党的精神传承犹如桂花飘香一样散发到各基层党员中，熏陶着我们、培育着我们。以下是我近段时间的思想汇报：

　　国家主席习近平9月3日在厦门国际会展中心出席金砖国家工商论坛开幕式，并发表题为《共同开创金砖合作第二个“金色十年”》的主旨演讲，强调金砖国家要共同开创金砖合作第二个“金色十年”，让合作成果惠及五国人民，让世界和平与发展的福祉惠及各国民众。

　　思想上，我们要加强认识，深化意识形态。我国已全面推进改革开放，社会经济在飞速的发展，作为一名入党积极分子（预备党员），思想上要有推动改革的决心，与国家发展基调保持一致，全面发力、多点突破，不断积聚经济持续发展的新动能。习主席指出：中国将深入贯彻创新、协调、绿色、开放、共享的发展理念。创新的思想要贯穿始终，改革发展离不开创新，不仅是技术的革新，最重要的思想要得到升华；开放与共享也是思想进化的添加剂，在“互联网+思维”中，共享已经变成一种生活常态，共享共赢是并存的，因此作为一名入党积极分子（预备党员），我们要在党的领导下拓宽思路，创新思维模式，为中国的经济做创新引领，为世界和平与发展作出新的更大贡献。

　　9月9日，央视网消息：“今天的学生就是未来实现中华民族伟大复兴中国梦的主力军，广大教师就是打造这支中华民族'梦之队'的筑梦人。”每逢教师节前夕，尊师重教的习近平总书记都会到学校看望这些筑梦人或致信表示祝贺，并对教育改革、教育公平、学生成长、人才培养和学校发展寄予殷切期望。

　　学习上，中华民族的传统美德及礼仪中重点突出“天地君亲师”。习总书记在每年的教师节都会去高校进行师德引路讲话。终身学习是人生的必修课程，我们必须把加强理论学习当作今后工作和生活中的日常任务。只有通过不断的学习充电，才能更好的适应社会发展的瞬息万变；只有不断的提高自己的思想政治素养，才能更好的为社会经济发展服务。实践出真知，专业技能的学习也需要我们以饱满的学习热情对待，以入党积极分子（预备党员）的身份展现积极的精神面貌，开展工作和学习，还要学会举一反三、融会贯通、活学活用，不能做思想上的巨人行动上的矮子，成为纸上谈兵，要能够把专业理论学习运用到实际工作中去，更好的为行业发展、社会进步做好服务建设工作。

　　与此同时，我们更应保持尊师重道的良好传统。虽不是教师，但是要延续老师的师德师风刻苦修炼、淡泊名利、言传身教，这样才能更好的成为入党积极分子（预备党员）行为规范的代表，具有丰富的知识和高尚的道德素养，同时也是同学（同事）学习的榜样，为党的精神进行切身实际的宣传。

　　工作上，作为一名入党积极分子（预备党员），我在各方面都严格要求自己，认真对待班级与学生会工作，平等、公平的对待服务的每一位同学（同事）。经常反省工作中的问题，在处理一些日常事务性的工作中，往往会受到自身约束和外部条件的影响，给自己的判断诱导进误区，呈动态性的变化；因此需要经常性的反思工作中的言行是否符合党的宗旨，是否一名合格共产党员的言行，增强群众意思、服务意识。同时要不断的向行业前辈学习，工作的经验要日积月累，在工作中汲取精华提升自我。

　　2017年以来，我在党组织、校（单位）领导的关心和帮助下取得了一定的成绩，但我知道还存在着一些缺点和不足，一是部门贡献度有所欠缺，几位优秀的党员一直是我学习的榜样，部门贡献度和他们比需要继续努力；二是实践能力较弱，理论知识与实践应用结合还不能无缝对接，在有些关键点还是需要领导提醒，没有完全做到自我解决问题。

　　十九大即将召开，在这个盛大的会议来临之际，我会进一步谨言慎行，以党员的高要求严标准做好自己，起到入党积极分子（预备党员）的模范带头作用，虚心向领导、老师及老党员学习，把缺点缩小变无，争取早日加入中国共产党，更好的接受党的精神洗礼，更好的为人民、为社会服务。请党组织考验我吧！

　　此致

敬礼！
