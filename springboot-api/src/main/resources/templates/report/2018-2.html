　　博山金暖一帘香，鳌山宫阙倚晴空；论情旋旋移相就，坛前乔木列仙家；利锁名缰空自苦，在地愿为连理枝；千乘万骑西南行，秋月春风等闲度。踏入春风拂面的第二季度，乘着十九大和十九届二中、三中全会精神，特将近期所学所感汇报如下：

　　海南博鳌，盛会天下。三角梅绽放，棕榈挺立，空气清爽沁人心脾。博鳌亚洲论坛2018年年会于4月8日拉开帷幕，迎接来自世界各国的2000多位嘉宾。年会以“开放创新的亚洲，繁荣发展的世界”为主题，分为“全球化与‘一带一路’”“开放的亚洲”“创新”和“改革再出发”四大板块，共60多场正式讨论。最受瞩目的是，中国国家主席习近平将应邀出席年会开幕式并发表重要主旨演讲，会上习总书记提出了十大金句振奋人心。

　　作为一名新时代的大学生，我们不仅要理解参透这十大金句，更要结合自身未来的发展，保持一颗向上的心，实现自己的人生价值。

　　一、学习方面，立足专业，始触科研

　　在学习好XX专业知识的同时，我也积极投身于各项学科竞赛。2017年9月参加全国大学生数学竞赛获江苏赛区一等奖、2016年5月参加校数学建模竞赛获二等奖、2016年9月参加全国大学生数学竞赛获江苏省三等奖。在2017、2018的寒假期间，我和队友连续两年参加了美国大学生数学建模比赛。（此部分结合自己实际情况写参加比赛）参加学科竞赛，它的意义远不止获得一张证书，更是对专业知识和能力检验、对学科前沿知识了解的机会，团队间的协作交流和学习更是终身受益。为了提高自己的科研能力，我主持了2017校级大学生创新训练项目《XXXXXXXXXXX》。为了保证项目的顺利开展，我和指导老师、小组成员进行了多次交流研讨，一边接受着专业知识的洗礼和熏陶，一边也享受着应用所学、解决问题的满足和喜悦。我将始终保持自己的求知欲，勇敢地接受挑战，深入一个又一个未知的领域，让科研成为人生中浓墨重彩的一笔。

　　二、思想方面，保持进步，紧随党走

　　作为院年级学生会XX部门成员，我更加严格要求自己，抓住学习机会，加强自身建设和党性修养，力争发挥出学生党员的先锋模范作用；作为校“先锋党校”学员，我见贤思齐、善于倾听，保质保量地完成团队中的每一项工作，上一年度获得了我校“优秀学生（党员、预备党员、入党积极分子）”的荣誉称号。作为班级生活委员，我严于律己、宽以待人，尽自己所能关心集体、服务同学、支持班级各项建设工作，并发挥自己在学习上的优势积极为同学们答疑解惑。在日常学习中，对于同学请教的问题我都会专心为其解答；期末前，从不吝啬自己整理的笔记和心得，分享给需要帮助的同学作为学习资料；新学期开始时，我也会给一些补考的同学提供复习建议和指导，使他们能够尽快的掌握核心知识点。希望借一己之力带领全班同学一起努力，风雨兼程一起进步，同学们的微笑和进步是对我最好的肯定，也是我莫大的荣幸。

　　三、生活中，全面发展，奉献社会

　　专业学习之余，投入社会实践活动也是一大乐事。2018年X月，我带领小组成员开展了“XXX”调研主题社会实践活动，并获得了新浪网等媒体（校园网）的报道。从起初对调查问卷进行设计和发放、对XX企业进行走访，与优秀毕业生学长的对话，到中期利用计算机各类软件对所得数据进行汇总、整理和分析并开展小组讨论，再到后期借助网络媒体对调研成果等进行宣传。虽只有一个月不到的时间，但我们却因付出辛劳和汗水而感到充实、因数据和结论的得出而感到欣慰。我们实现了善用理论知识、增加社会经验、提高实践能力、丰富假期生活的共赢。苦中作乐，不亦乐乎？

　　在志愿服务上，我也积极投入、乐在其中。“XX小学义务支教”活动、“XX敬老院爱心之行”活动、“保护湿地、留住诗意”主题活动等都让我体会到了“授人玫瑰、手留余香”、“自助助人”的感动、感恩和幸福，更坚定了人生就应该不断奉献、不断给予，用自己的能力去帮助他人、服务社会的想法。多彩的志愿服务，丰富了我的课余生活，锻炼了我的实践能力，增加了我的社会经验，也是我追梦之路上的亮丽风景线。
