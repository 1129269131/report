思想汇报

尊敬的党组织：

　　寒梅傲立，冬日暖阳，我们被党的光辉普照着、温暖着、呵护着。回首2017年，如层林尽染，从十八大的成果丰收到十九大的推新开局，中国共产党的精神文化万山红遍，指引着每一代人。现将近期的思想升华作如下汇报：

　　中共中央12月6日在中南海召开党外人士座谈会，就今年经济形势和明年经济工作听取各民主党派中央、全国工商联负责人和无党派人士代表的意见和建议。中共中央总书记习近平主持座谈会并发表重要讲话。习近平强调，2018年是贯彻中共十九大精神的开局之年，是实施“十三五”规划承上启下的关键一年。经济工作要贯彻新理念、聚焦新目标、落实新部署，为2020年全面建成小康社会打下坚实物质基础。

　　一、贯彻新理念，坚持文化自信

　　本次虽是党外人士座谈会，但对我们入党积极分子（预备党员）具有超强的指导意义。在我们通向党员与连接群众的道路上，要把新思想与新理念“学准吃透”，坚持文化自信，贯彻新理念，把党的宗旨、精神文化等推广到群众中去，在群众中生根发芽。

　　在思想上，文化自信不是口号。首先，我们的文化自信体现在理论方面，坚持科学社会主义基本原则。例如，坚持把实现共产主义作为最高理想，坚持人民群众的主体地位，坚持实事求是的原则等。这些都是科学社会主义基本原则在新的历史条件下的具体表现。

　　其次，在文化自信制度保障的实践方面，我国实行人民代表大会制度，保证了人民当家做主的权利，极大地调动了人民的积极性和创造性，使我国的综合国力大大提升。

　　另外，我们的文化自信还体现在创新上，文化自信并不是守着自己现有的成就骄傲自满，而是随着时代的发展，在进一步完善和发展中国特色社会主义政治制度的进程中，增强民族文化的创新性。改革开放以来，中国推进政治体制改革，实现了民生的改善，权力腐败的改善，促进了社会的和谐。

　　二、聚焦新目标，开启学习新征程

　　十九大会议学习后，我们入党积极分子（预备党员）应以习近平新时代中国特色社会主义思想为指导，坚持以学习为中心，进一步加强专业学习，强化自身行业新知识，优化自身专业新技能，提高管理服务水平，努力提升自我学习意识，丰富文化学习内涵，促进自身专业健康有序发展，努力达成高素质技能型人才的目标。

　　在学习上，与同学之间要本着优势互补、资源共享、相互促进、共同发展的原则，加强沟通交流，发挥协作优势，通过深入合作，聚焦学习新目标，开启学习新征程。各行业都面临着人才匮乏的现象，但同时也有人员淘汰裁剪的现象，我们入党积极分子（预备党员）必须要站在市场前沿，以新目标、新学习为切入点，进一步扩大学习范围，积极探索行业新知识，成为国家和企业新知识高素质储备人才。

　　三、落实新部署，提升服务能力

　　习近平总书记说：干部就要有担当，有多大担当才能干多大事业，尽多大责任才会有多大成就。我们入党积极分子（预备党员）在工作中要与普通群众有所区别，因为我们的责任更大、担子更重，凡是要做表率，遇到困难永远要冲在群众的前面，同时树立先锋形象，着力营造入党积极分子（预备党员）优良作风。

　　在工作中，我十分重视本职工作，作为一名学生会XX部门成员，我很注意同学的思想政治建设，经常深入到同学中去，零距离倾听同学的呼声，了解他们的思想状况，加强与广大同学的交流与沟通，解决他们的思想困惑和实际问题；重视把党的精神文化宣传渗透到各种场合,增进同学们的认同感，创造团结和谐、积极向上的局面。

　　同时落实本职工作的新部署任务，提升服务能力。加强团队的建设与管理，完善“团队、平台、项目”有机结合的管理和运行机制，联合相关部门合作开展服务工作，提升部门的服务能力。推动XX部门工作的良性互动协调发展，着力构建XX专业知识对XX专业技能的支撑与反哺机制，以知识带技能，以实际应用促发展，保障XX行业工作持续稳定发展。

　　在向党组织靠拢的过程中，作为一名入党积极分子（预备党员）我不断交流、学习，发挥先进模范作用，将所学知识与党员前辈及老师的经验积累连接起来，实现全方位、多层次的自身定位与规划，在今后我将摒弃不良习惯，进一步完善自我，争取早日加入中国共产党，更好的为全面建成小康社会做好服务。请党组织继续考验我吧！

　　此致

敬礼！
